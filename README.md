# 💬 Paschala Hub
### A Real-Time WhatsApp Clone for Seamless Communication

Paschala Hub is a lightweight, mobile-optimized chat application designed for instant messaging. Built by **Azubuike Paschal Kamsiyochukwu**, this project focuses on speed, real-time updates, and a clean user experience using a modern web stack.

## 🌟 Key Features
* **Instant Real-time Chat**: Powered by Firebase Firestore for zero-latency messaging.
* **Google One-Tap Auth**: Secure and fast user authentication using Firebase Auth.
* **Mobile-First UI**: Responsive design tailored for mobile screens using Tailwind CSS.
* **Clean Architecture**: Built with Vite for ultra-fast development and build times.

## 🛠️ The Tech Stack
* **Frontend**: React (Vite)
* **Styling**: Tailwind CSS & Lucide Icons
* **Backend**: Firebase (Auth & Firestore)
* **Deployment**: Vercel
